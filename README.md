# Concepts Builder

```
This Project is for generating concept-topic tables from a given chapter.
This is being done to increase the syllabus coverage of our product 'qgen'
```

## Flow

```
There will be one dedicated prompt per book or per book format.
All chapters in a single book generally follow the same format, so that can use the same prompt.
Multiple books which share the same format will use the same prompt.
```

## Input

```
Input will be a chapter's PDF
```

## Output

```
Output will be a csv file containing concept-topic table
```
